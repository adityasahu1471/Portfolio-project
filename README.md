# Portfolio Website

A personal portfolio website built from scratch using HTML and CSS.

## 🔗 Live Demo
[View Portfolio](https://adityasahu1471.github.io/Portfolio-project/)

## About:
Hi, I'm Aditya, a student at Amrita University. This portfolio was built as part of a UI/UX assignment. I used knowledge from PowerPoint lessons, W3Schools, YouTube tutorials, and my own research to design and build this project.

## Built With:
- **HTML5** — page structure and semantic markup
- **CSS3** — styling, layout, animations, and responsive design
- **Google Fonts** — Poppins and Merriweather
- **Font Awesome 7** — icons for navigation and section links

## Things I Learned from doing the project:
- Structuring an HTML page with semantic sections
- Linking external CSS and Google Fonts
- Using Flexbox for layout and alignment
- Creating hover effects and smooth transitions
- Using `backdrop-filter: blur` for frosted glass UI
- Applying radial gradients for background design
- Making websites responsive using media queries


## A Step by Step journal :


### Setup
- Created `index.html` and `style.css` and linked them together
- Added Google Fonts (Poppins, Merriweather) and Font Awesome for icons

### Section 1 — Navigation
- Header with "My Portfolio" on the left
- Social links (GitHub, LinkedIn, Email) on the right using Flexbox
- Links open in a new tab using `target="_blank"`
- CSS: utility classes for flex, gap, padding, margin, and color
- Background uses a multi-layered radial gradient (colors researched online)
- At 600px width, link text hides and only icons show

### Section 2 — Profile
- Name, bio (lorem ipsum placeholder), and profile image
- Quick-link buttons for Skills, Previous Projects, and Contact
- Hover effect brightens the buttons
- Image styled with width, height, border-radius, and drop-shadow

### Section 3 — Skills
- Cards for HTML, CSS, JavaScript, Node.js, and React.js
- Frosted glass effect using `backdrop-filter: blur`
- Flexbox used to center and wrap cards
- Hover lifts each card with a subtle shadow

### Section 4 — Contact Form
- Form with Name, Topic, and Message fields
- Styled inputs with transparent background, blur, and border
- Submit button with hover lift effect

### Section 5 — Responsive Design
- **850px** — reduced heading font size
- **740px** — profile section stacks vertically (image on top)
- **600px** — nav link text hidden, only icons visible; padding reduced
